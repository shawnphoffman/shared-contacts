# Shared Contacts UI

Frontend for the Shared Contacts app. Built with TanStack Router + Query, Vite,
and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:3030`.

## Build + Preview

```bash
npm run build
npm run start
```

## Lint + Format

```bash
npm run lint
npm run format
```
