# Shared Contacts UI

Frontend for the Shared Contacts app. Built with TanStack Router + Query, Vite,
and Tailwind CSS.

## Local Development

```bash
pnpm install
pnpm dev
```

The dev server runs on `http://localhost:3030`.

## Build + Preview

```bash
pnpm build
pnpm start
```

## Lint + Format + Test

```bash
pnpm lint
pnpm format
pnpm test
```
